Printing/Downloading this tutorial to PDF

You can print (e.g. to PDF file) your current processing state of this tutorial. To this end, you have to open the tutorial within a standard web browser like Firefox, Chrome, Safari, Edge, ... That is, while you are running the tutorial within RStudio, you can use the `Open in Browser` button in the upper left corner of the "Viewer" window that shows the tutorial. Within the browser, just use the standard printing option, e.g. using the "CTRL+P" keyboard shortcut.
